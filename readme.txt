 == Description ==

 Shiga is a nice Custom Login Page for WordPress.

 To make your WordPress Login (wp-login.php) page look nicer - with a nice layout.

 == Installation ==

 1. Upload the plugin files to the `/wp-content/plugins/shiga-custom-login-by-corelabs` directory, or install the plugin through the WordPress plugins screen directly.
 2. Activate the plugin through the 'Plugins' screen in WordPress
 3. Use the Settings->Shiga Custom WP Login screen to configure the plugin
 4. Select or Upload an image for the login page
 5. Select a background color using the color picker tool
 6. Save changes and logout to preview (or preview in another browser)


 == Frequently Asked Questions ==

 = My image is not showing =

 Check that your Logo URL is valid and opens an image when entered into a browser.

 = How do I get a Hexadecimal value for color =

 You can use any color picker tools to choose your color.
